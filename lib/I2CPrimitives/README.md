# I2CPrimitives

**Purpose:** Small **Arduino `Wire`** helpers: write, read, and write-then-read (register + data).

**Primary files:** `primitives.h`, `primitives.cpp`

**API:** `i2c_write()`, `i2c_read()`, `i2c_write_read()`

**Used by:** Mux driver and other Teensy I2C callers on the shared bus.

**Documentation:** [Library index](../../docs/libraries/index.md)
